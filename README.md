## üöÄ Getting Started

### Prerequisites
- Node.js (Latest LTS)
- Firebase Project setup
- Azure AI Services credentials

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/infralith.git
    cd infralith
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Configure Environment**:
    Create a `.env.local` file with your credentials (refer to `.env.example`).

4.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:3500`.
